# regression
Обучение моделей методами Линейной регрессии, LASSO, Ridge

Регрессия. Под регрессией понимается семейство методов машинного обучения, предназначенных для прогнозирования непрерывных числовых значений на основе входных признаков.

Задача — построить модель, которая количественно описывает взаимосвязь между независимыми переменными и целевым показателем. Качество модели определяется её способностью точно предсказывать значения на новых данных.

В работе успешно проведен полный цикм машинного обучения: от детального анализа данных и предобработки (удаление пропусков, масштабирование) до применения и сравнительной оценки трех алгоритмов регрессии. Выполнен подбор оптимальных гиперпараметров для регуляризованных моделей с использованием кросс-валидации.

По результатам сравнения метрик качества, модель линейной регрессии без регуляризации продемонстрировала результаты, сопоставимые с регуляризованными моделями (LASSO и Ridge), что указывает на ограниченность линейных зависимостей в данных и отсутствие выраженной мультиколлинеарности. Для повышения точности прогнозов целесообразно рассмотреть расширение признакового пространства или применение нелинейных методов
